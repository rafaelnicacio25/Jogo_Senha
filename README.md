### **NOME DO JOGO:**
## Enigma das Ruínas

## História do Jogo
O jogo se passa em um período medieval místico, onde elfos, dragões e tesouros lendários estão espalhados pelo mundo. Neste contexto, o protagonista está em busca de um tesouro antigo que, para muitos, não passava de uma lenda. Ao vasculhar ruínas devastadas, ele encontra **Lily**, uma criatura misteriosa e animada que te auxiliará (ou não) na sua jornada!

## Como Executar o Jogo

O jogo é bem simples de ser executado, o usuário deve ter algum compilador em C para rodar o jogo, após ter o compilador, o usuário deve baixar o arquivo ou copiar o código e executá-lo.
